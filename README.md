# 🏦 Bank Management System

A desktop-based Bank Management System built using Python (Tkinter) and MySQL.

## 🚀 Features

- Open Account
- Deposit Money
- Withdraw Money
- Balance & Mini Statement
- Transaction History

## 🛠 Technologies Used

- Python
- Tkinter
- MySQL
- PyMySQL

## 📌 How to Run

1. Install requirements:
   pip install -r requirements.txt

2. Setup MySQL database:
   Create database: bankdb

3. Run:
   python bank.py
