# 🏦 Bank Management System

A desktop-based Bank Management System built using Python (Tkinter) and MySQL.

## 🚀 Features

- Open Account
- Deposit Money
- Withdraw Money
- Balance & Mini Statement
- Transaction History

## 🛠 Technologies Used

- Python
- Tkinter
- MySQL
- PyMySQL

## 📌 How to Run

## 📦 Installation & Setup

Follow these steps to run the project on your system:

### 1️⃣ Clone the Repository

git clone https://github.com/Dhaval-Solanki11/bank-management-system.git

### 2️⃣Install Required Library

pip install pymysql

3️⃣ Setup MySQL Database

Open MySQL Workbench and run:
CREATE DATABASE bankdb;
USE bankdb;

Then import the provided file:
bankdb.sql

5️⃣ Run the Application

python Bank.py
